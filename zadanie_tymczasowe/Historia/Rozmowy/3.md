### `src/backend/storage/keyDir.ts` — Bitcask KeyDir RAM Index

```typescript
// ================================================================
// NEXUS V2 — KeyDir: Bitcask In-Memory Hash Index (Phase 2.1)
// ================================================================
// Paradygmat Bitcask (Log-Structured Hash Table):
// - Append-Only: brak nadpisywania, każda modyfikacja = nowy rekord
// - Tombstones: usunięcie = zapis znacznika śmierci, usunięcie z Map
// - O(1) RAM: pamięć trzyma tylko Map<id, DocumentMeta> — koordynaty
//   na dysku (filePath, offset, size). Żadne logi nie są ładowane do RAM.
// - Odczyt: precyzyjne wycięcie bajtów z pliku (handle.read z offsetem)
// - Zero-Copy zgodność: brak msgpackr, natywny JSON + Buffer
// ================================================================

import { promises as fs } from 'fs';
import path from 'path';

export interface DocumentMeta {
  filePath: string;
  offset: number;
  size: number;
  timestamp: number;
  isTombstone: boolean;
}

export class KeyDirError extends Error {
  constructor(message: string) {
    super(message);
    this.name = 'KeyDirError';
  }
}

export class DeletedObjectError extends KeyDirError {
  constructor(id: string) {
    super(`Deleted Object Found: "${id}" — rekord został logicznie usunięty (tombstone)`);
    this.name = 'DeletedObjectError';
  }
}

interface LogRecord {
  id: string;
  timestamp: number;
  tombstone?: boolean;
  data?: unknown;
}

export class KeyDir {
  private readonly map = new Map<string, DocumentMeta>();
  private readonly logFile: string;

  constructor(options: { logDir: string; logFileName?: string }) {
    this.logFile = path.join(options.logDir, options.logFileName ?? 'nexus.bitcask.jsonl');
  }

  // ============================================================
  // write(id, data) — Append-Only write + O(1) RAM update
  // ============================================================
  public async write(id: string, data: unknown): Promise<DocumentMeta> {
    const timestamp = Date.now();
    const record: LogRecord = { id, timestamp, data };
    const line = JSON.stringify(record) + '\n';
    const encoded = Buffer.from(line, 'utf8');
    const size = encoded.byteLength;

    let offset = 0;
    try {
      const stat = await fs.stat(this.logFile);
      offset = stat.size;
    } catch { /* plik nie istnieje — offset = 0 */ }

    await fs.appendFile(this.logFile, encoded, { flag: 'a' });

    const meta: DocumentMeta = { filePath: this.logFile, offset, size, timestamp, isTombstone: false };
    this.map.set(id, meta);
    return meta;
  }

  // ============================================================
  // get(id) — O(1) lookup → precyzyjne wycięcie bajtów z dysku
  // ============================================================
  public async get(id: string): Promise<unknown | null> {
    const meta = this.map.get(id);
    if (!meta) return null;
    if (meta.isTombstone) throw new DeletedObjectError(id);

    const handle = await fs.open(meta.filePath, 'r');
    try {
      const buffer = Buffer.alloc(meta.size);
      const { bytesRead } = await handle.read(buffer, 0, meta.size, meta.offset);
      if (bytesRead !== meta.size) {
        throw new KeyDirError(
          `KeyDir.get("${id}"): odczytano ${bytesRead}B zamiast ${meta.size}B`
        );
      }
      const parsed = JSON.parse(buffer.toString('utf8'));
      return parsed.data ?? null;
    } finally {
      await handle.close();
    }
  }

  // ============================================================
  // delete(id) — Tombstone: append znacznika śmierci
  // ============================================================
  public async delete(id: string): Promise<DocumentMeta> {
    const existingMeta = this.map.get(id);
    if (!existingMeta) {
      return { filePath: this.logFile, offset: 0, size: 0, timestamp: Date.now(), isTombstone: true };
    }
    if (existingMeta.isTombstone) return existingMeta;

    const timestamp = Date.now();
    const record: LogRecord = { id, timestamp, tombstone: true };
    const line = JSON.stringify(record) + '\n';
    const encoded = Buffer.from(line, 'utf8');
    const size = encoded.byteLength;

    const stat = await fs.stat(this.logFile);
    const offset = stat.size;
    await fs.appendFile(this.logFile, encoded, { flag: 'a' });

    const meta: DocumentMeta = { filePath: this.logFile, offset, size, timestamp, isTombstone: true };
    this.map.set(id, meta);
    return meta;
  }

  public has(id: string): boolean {
    const meta = this.map.get(id);
    return !!meta && !meta.isTombstone;
  }

  public size(): number {
    let count = 0;
    for (const meta of this.map.values()) { if (!meta.isTombstone) count++; }
    return count;
  }

  public entries(): IterableIterator<[string, DocumentMeta]> {
    return this.map.entries();
  }

  public getLogFile(): string { return this.logFile; }
  public setFromWorker(id: string, meta: DocumentMeta): void { this.map.set(id, meta); }
  public async loadFromHintFile(_hintPath: string): Promise<number> { return 0; }
}
```

## 3. 🧠 SYSTEM SURVIVAL — 3 powody, dla których ten kod nie wywali aplikacji

1. **Precyzyjny wycinek dysku (`Buffer.alloc(meta.size)`)** — odczytuje DOKŁADNIE tyle bajtów, ile miał oryginalny rekord. Nawet jeśli zewnętrzny proces dopisze śmieci na koniec pliku (Test 1: corruption proof), KeyDir wycina stary, niezmieniony fragment. Zero ryzyka parsowania śmieci → zero crashy JSON.parse.

2. **Tombstone z `DeletedObjectError`** — usunięcie rekordu to append tombstone + ustawienie `isTombstone: true` w indeksie. Próba `get()` na tombstone rzuca twardy błąd domenowy (`DeletedObjectError`), a nie zwraca `null` — aplikacja nie może przypadkiem operować na martwych danych. `has()` zwraca `false`, więc żaden proces nie zobaczy ghost-rekordu.

3. **Brak phantom tombstone w Map** — `delete()` na ID, które nigdy nie istniało, **nie dodaje** wpisu do mapy. `get()` na takim ID zwraca `null` przez `!meta`, a nie przez `isTombstone`. Eliminuje to całą klasę błędów, gdzie pusta mapa kumuluje martwe wpisy. `delete()` jest idempotentny — drugi delete na tombstone zwraca istniejące meta bez zapisu na dysk.
