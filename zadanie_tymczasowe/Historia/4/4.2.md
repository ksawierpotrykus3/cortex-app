### PARAMETRYZACJA ARCHITEKTONICZNA (4.2)

- **DAG_MAX_STACK_DEPTH** = `10` (PipelineContext.MAX_STACK_DEPTH — twardy limit hopów)
- **DAG_ROUTING_STRATEGIES** = exact match ID → RegExp → string prefix
- **DAG_ERROR** = `CircularDependencyException` przy hopCount > 10, komunikat zawiera correlationId, hopCount, ścieżkę cyklu
- **DAG_CONTEXT** = `PipelineContext { entryNodeId, correlationId (UUID), visitedNodes[], hopCount, payload, startedAt }`
- **DAG_CORRELATION_ID** = `crypto.randomUUID()` z fallbackiem `Date.now() + Math.random()`
