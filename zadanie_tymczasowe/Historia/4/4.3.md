### WERYFIKACJA DETERMINISTYCZNA (TDD)

- **ModelCache:** ModelNotFoundError przy braku pliku, za mały plik (< 1MB), poprawny plik (≥ 2MB), listModels()
- **Negacja wink-nlp:** "no network" → `network.negationFlag = true`. "working internet connection" → wszystkie `negationFlag = false`
- **LexicalEngine:** Indeksacja + wyszukiwanie BM25, filtracja negacji, analyzeQuery rozdziela positive/negated
- **Offline test:** ModelCache nie próbuje łączyć się z HuggingFace — brak sieci = błąd z komunikatem

### PARAMETRYZACJA ARCHITEKTONICZNA (4.3)

- **MODEL_CACHE_DIR** = `models/` (w katalogu projektu, resolvany jako pierwszy istniejący)
- **MODEL_CACHE_MIN_SIZE** = `1_000_000` (minimalny rozmiar pliku .onnx w bajtach — ~1MB)
- **ONNX_SINGLETON** = `OnnxRuntime.getInstance()` — model all-MiniLM-L6-v2.onnx ładowany DOKŁADNIE RAZ
- **ONNX_EXECUTION_PROVIDER** = `['cpu']` — zero GPU, zero sieci
- **ONNX_GRAPH_OPTIMIZATION** = `'all'` (`graphOptimizationLevel`)
- **ONNX_MAX_TOKENS** = `256` (maxTokens dla tokenizera)
- **ONNX_EMBEDDING_DIMS** = `384` (wymiar wektora all-MiniLM-L6-v2)
- **ONNX_TOKENIZER** = WordPiece hash: [CLS]=101 → hash słów → [SEP]=102, zakres [103, 29103]
- **ONNX_MEAN_POOLING** = dla `last_hidden_state` [1, seq_len, 384] → średnia po seq_len
- **LEXICAL_BM25_PARAMS** = `{ k: 100, k1: 1.2, b: 0.75, norm: 'l2' }`
- **LEXICAL_WINK_PIPES** = `['sbd', 'negation', 'pos', 'ner', 'sentiment']`
- **LEXICAL_NEGATION_FILTER** = dokumenty z zanegowanymi terminami odrzucane (`hasNegatedTerm → continue`)
- **LEXICAL_MODEL** = `wink-eng-lite-web-model` (angielski model NLP, ~5MB, działa offline)
